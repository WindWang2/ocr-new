# Demo 仪器图片与类别映射说明

本文档记录了系统中支持的 9 种仪器类型及其对应的 YOLO 类别 ID（0-8）。
**注意：以后仪器统一称为 D0-D8；相机统一称为 F0-F8。**

## YOLO 类别与仪器映射 (D0-D8)

系统采用 YOLO 进行目标检测与分类，类别 ID 直接对应仪器编号。

| 类别 ID | 内部标识 | 仪器名称 | 物理相机 (F) |
| :--- | :--- | :--- | :--- |
| **0** | **D0** | **吴英混调器** | F0 |
| **1** | **D1** | **电子天平 1 号** | F3 |
| **2** | **D2** | **电子天平 2 号** | F3 |
| **3** | **D3** | **PH 仪** | F3 |
| **4** | **D4** | **水质检测仪** | F5 |
| **5** | **D5** | **表界面张力仪** | F5 |
| **6** | **D6** | **电动搅拌器** | F7 |
| **7** | **D7** | **水浴锅** | F7 |
| **8** | **D8** | **6 速旋转粘度计** | F8 |

## 样例图片说明

原始测试样本存放在 `demo_images/` 目录下（BMP 格式）。

### 典型样本参考：
- **im001.bmp**：对应类别 7 (D7 水浴锅)，LED 数码管显示。
- **im005.bmp**：对应类别 3 (D3 PH 仪)，LCD 背光屏。
- **im009.bmp**：对应类别 1/2 (D1/D2 天平)，LCD 液晶屏。
- **im0014.bmp**：**复合场景**，包含多个仪器（如 D4 水质检测仪和 D5 张力仪）。

## 识别逻辑

1. **检测 (Detection)**：YOLO 在全景图中定位所有仪器，输出 bbox 和类别 ID。
2. **裁剪 (Cropping)**：系统根据 bbox 自动抠出 `Padding=15px` 的特写图。
3. **识别 (Reading)**：调用大模型根据类别 ID 匹配对应的 Prompt 模板，提取结构化读数。
