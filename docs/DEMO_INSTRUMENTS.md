# Demo 仪器图片与类别映射说明

本文档记录了系统中支持的 9 种仪器类型及其对应的 YOLO 类别 ID（0-8）。

## YOLO 类别与仪器映射 (F0-F8)

系统采用 YOLO26x 进行目标检测与分类，类别 ID 直接对应原有的相机位编号。

| 类别 ID | 内部标识 | 仪器名称 | 关键检测字段 |
| :--- | :--- | :--- | :--- |
| **0** | `wuying_mixer` | **超级吴英混调器** | 转速、时间、模式、剩余时间 |
| **1** | `electronic_balance` | **电子天枰 1 号** | 重量 (weight) |
| **2** | `electronic_balance` | **电子天枰 2 号** | 重量 (weight) |
| **3** | `ph_meter` | **PH 仪** | pH值、温度、PTS值 |
| **4** | `water_quality_meter` | **水质检测仪** | 含量、透光度、吸光度、检测值 |
| **5** | `surface_tension_meter`| **表界面张力仪** | 张力、温度、密度、升降速度 |
| **6** | `torque_stirrer` | **电动搅拌器** | 转速、扭矩、时间 |
| **7** | `temperature_controller`| **水浴锅** | 温度、定时时间 |
| **8** | `viscometer_6speed` | **6 速旋转粘度计** | 粘度、转速、剪切率、剪切应力 |

## 样例图片说明

原始测试样本存放在 `demo_images/` 目录下（BMP 格式）。

### 典型样本参考：
- **im001.bmp**：对应类别 7 (水浴锅)，LED 数码管显示。
- **im005.bmp**：对应类别 3 (PH 仪)，LCD 背光屏。
- **im009.bmp**：对应类别 1/2 (天平)，LCD 液晶屏。
- **im0014.bmp**：**复合场景**，包含多个仪器（如 F4 水质检测仪和 F5 张力仪）。

## 识别逻辑

1. **检测 (Detection)**：YOLO26x 在全景图中定位所有仪器，输出 bbox 和类别 ID。
2. **裁剪 (Cropping)**：系统根据 bbox 自动抠出 `Padding=15px` 的特写图。
3. **识别 (Reading)**：调用大模型根据类别 ID 匹配对应的 Prompt 模板，提取结构化读数。

## 历史兼容性说明
旧版的 `demo/` 文件夹（JPG 格式）中的图片名（如 `1.jpg`）与现在的类别 ID 可能不完全一致，建议优先参考 `demo_images/` 中的 BMP 样本进行调试。
